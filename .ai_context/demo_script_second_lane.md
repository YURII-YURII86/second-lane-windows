# Second Lane Demo Script

## Goal

Record a short demo that proves the core promise:

When Claude Code or Codex stops, Second Lane lets you continue the task inside ChatGPT with real project actions on your machine.

## Recommended Demo Length

- 20 to 45 seconds

## Demo Structure

### 1. Set the pain

Show a short on-screen context line:

`Claude Code hit a limit before the task was done.`

or

`Codex stopped, but the project still had to move forward.`

### 2. Move into ChatGPT

Show:

- ChatGPT open
- the Custom GPT selected
- the task continuing from the previous state

### 3. Show real hands

Recommended action sequence:

1. `inspectProject`
2. read the relevant file
3. apply or verify a patch flow
4. run a command or test
5. show the successful result

### 4. Close on payoff

End with a visible success state and a line like:

`The task kept moving inside ChatGPT.`

## What The Viewer Must Understand

By the end of the demo, a new visitor should understand:

- this is not just chat;
- ChatGPT is actually acting on the project;
- the tool is useful when another coding agent hits a limit.

## Recording Notes

- keep zoom readable
- avoid private secrets on screen
- keep the terminal output short
- prefer one concrete task over many micro-actions
